Triple DES Encryption

This tool uses the Triple DES algorithm to encrypt plaintext into ciphertext, and vice versa.

To use the tool for encryption, enter the plaintext you want to encrypt, and any keys that you want to use
(You may leave the keys blank to use random keys).

To use the tool for decryption, leave the Plaintext field blank, and enter the keys and ciphertext that you want to decrypt.

Note that if the Plaintext and Ciphertext fields are left blank, the tool will simply encrypt the empty string, which will
result in another empty string.
